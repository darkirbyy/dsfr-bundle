# Form

## Dépendances

Aucune

## Installation

Pour utiliser le form theme DSFR (récupéré de packagist), il faut ajouter la configuration suivante dans `twig.yaml` :

```yaml
# config/packages/twig.yaml
twig:
  [...]
  form_theme: '@DarkirbyDsfr/form/theme.html.twig'
```

Automatique
